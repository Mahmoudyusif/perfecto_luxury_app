# Perfecto Luxury 👗 - Full-Stack E-commerce Suite

Perfecto is a premium, high-performance Flutter application designed for luxury fashion brands. It provides a seamless shopping experience for customers and a comprehensive, hidden management system for brand owners.

## 🌟 Key Highlights
- **Architecture:** Built with a focus on speed using `IndexedStack` for near-instant navigation.
- **Backend:** Fully integrated with **Firebase** (Firestore, Cloud Messaging, and Auth).
- **Dual-Mode System:** A single app serves both the customer (B2C) and the owner (Admin Dashboard) via a secure, hidden entry point.

## 🚀 Professional Features

### 🛍 For Customers (Elite Experience)
- **Elite Membership:** Automated loyalty points system that rewards repeat purchases.
- **Dynamic Offer Bar:** An auto-scrolling announcement bar controlled in real-time by the admin.
- **Advanced Search:** Real-time product discovery across the entire catalog.
- **Wishlist & Personalization:** Save favorites and manage profiles with session persistence.
- **Review System:** Real-time product ratings and customer feedback.
- **Secure Checkout:** Multi-method payment support with dynamic Promo Code validation.

### 💼 For Brand Owners (The Command Center)
- **Financial Analytics:** Real-time tracking of daily and monthly revenue.
- **Live Inventory Manager:** Add, edit, or delete products and toggle "Out of Stock" status instantly.
- **Broadcast Notifications:** Send Push Notifications to all app users directly from the dashboard.
- **Order Pipeline:** Full tracking and management of shipments (Processing -> Shipped -> Delivered).
- **Branch Management:** Dynamic control over store locations and contact information.

## 🛠 Technical Stack
- **Framework:** Flutter
- **State Management:** Provider & ChangeNotifier (Reactive UI)
- **Database:** Google Cloud Firestore (NoSQL)
- **Messaging:** Firebase Cloud Messaging (FCM)
- **Local Storage:** SharedPreferences (Session persistence)
- **Design:** Luxury UI/UX with Google Fonts (Cairo) and optimized assets.

## 📸 How to Access Admin Mode
For security, the Admin Dashboard is hidden. To access:
1. Navigate to the **Account** tab.
2. **Long Press** the profile avatar.
3. Enter the secure access key (Default: `1234`).
